* cl-llama-cpp

THIS IS A WORK IN PROGRESS! DO NOT USE!

Common Lisp bindings to [[https://github.com/ggml-org/llama.cpp][llama.cpp]] via CFFI.

Provides a high-level, Lisp-idiomatic interface for loading GGUF models, tokenization, text generation, and embeddings, with automatic resource management.

** Features

- Load and run GGUF models (CPU and GPU)
- Tokenize and detokenize text
- Generate text with configurable sampling (temperature, top-k, top-p, min-p, greedy)
- Compute text embeddings
- Automatic resource cleanup via =with-= macros
- Proper condition hierarchy for error handling
- Bindings auto-generated from =llama.h= using [[https://github.com/borodust/claw][CLAW]]

** Requirements

- *SBCL* (required — llama.cpp triggers floating-point exceptions that only SBCL's =sb-int:with-float-traps-masked= handles correctly)
- [[https://github.com/ggml-org/llama.cpp][llama.cpp]] built as a shared library (=libllama.so= / =libllama.dylib=)
- [[https://cffi.common-lisp.dev/][CFFI]] and [[https://github.com/cffi/cffi][cffi-libffi]]

** Installation

*** Building llama.cpp

The repository includes llama.cpp as a git submodule.

#+begin_src bash
git clone --recursive https://github.com/licjon/cl-llama-cpp
cd cl-llama-cpp/llama.cpp
cmake -B build -DBUILD_SHARED_LIBS=ON
cmake --build build -j$(nproc)
#+end_src

The library loader will automatically find =libllama.so= in =llama.cpp/build/bin/=. Alternatively, install it somewhere on your system library path.

*** Loading with ASDF

Place or symlink this directory into your ASDF source registry (e.g. =~/common-lisp/=), then:

#+begin_src lisp
(asdf:load-system "cl-llama-cpp")
#+end_src

For Roswell users, clone into =~/.roswell/local-projects/= and it will be found automatically.

If you use Quicklisp and want =ql:quickload= to find it, run =ql:register-local-projects= first.

** Quick Start

#+begin_src lisp
(asdf:load-system "cl-llama-cpp")

(cl-llama-cpp:with-model (model "/path/to/model.gguf" :n-gpu-layers 99)
  (cl-llama-cpp:with-context (ctx model :n-ctx 2048)
    (princ (cl-llama-cpp:generate ctx "The capital of France is"
                                  :max-tokens 64
                                  :temp 0.7))))
#+end_src

** API Reference

*** Resource Management

- =with-model (var path &rest params) &body body= --- Load a GGUF model, bind to =var=, free on exit. Keyword params override =llama_model_default_params= (e.g. =:n-gpu-layers 99=). Signals =model-load-error= on failure.

- =with-context (var model &rest params) &body body= --- Create an inference context from =model=, bind to =var=, free on exit. Keyword params override =llama_context_default_params= (e.g. =:n-ctx 2048=). Signals =context-creation-error= on failure.

- =with-sampler-chain (var &key temp top-k top-p min-p seed greedy) &body body= --- Build a sampler chain, bind to =var=, free on exit.

*** Operations

- =tokenize model text &key add-special parse-special= --- Tokenize =text= into a vector of token integers.

- =detokenize model tokens &key remove-special unparse-special= --- Convert a token vector back to a string.

- =generate ctx prompt &key max-tokens temp top-k top-p min-p seed= --- Generate text continuing =prompt=. Returns the generated string. Blocks until EOS or =max-tokens=.

- =embed ctx text &key normalize= --- Compute embeddings for =text=. Returns a vector of single-floats. The context must be created with =:embeddings 1=.

*** Conditions

All conditions inherit from =llama-error=.

| Condition              | Slot | When                          |
|                        |      |                               |
|------------------------+------+-------------------------------|
| =model-load-error=       | =path= | Model file cannot be loaded   |
| =context-creation-error= |      | Context allocation fails      |
| =tokenization-error=     | =text= | Tokenization fails            |
| =decode-error=           | =code= | =llama_decode= returns non-zero |

** Examples

*** Tokenization

#+begin_src lisp
(cl-llama-cpp:with-model (model "/path/to/model.gguf" :n-gpu-layers 0)
  (let* ((tokens (cl-llama-cpp:tokenize model "Hello, world!"))
         (text   (cl-llama-cpp:detokenize model tokens)))
    (format t "Tokens: ~A~%Round-trip: ~A~%" tokens text)))
#+end_src

*** Embeddings

#+begin_src lisp
(cl-llama-cpp:with-model (model "/path/to/embedding-model.gguf" :n-gpu-layers 0)
  (cl-llama-cpp:with-context (ctx model :n-ctx 512 :embeddings 1 :pooling-type 1)
    (let ((vec (cl-llama-cpp:embed ctx "Hello, world!")))
      (format t "~D dimensions~%" (length vec)))))
#+end_src

*** Custom Sampling

#+begin_src lisp
(cl-llama-cpp:with-model (model "/path/to/model.gguf")
  (cl-llama-cpp:with-context (ctx model :n-ctx 2048)
    ;; Greedy decoding
    (cl-llama-cpp:with-sampler-chain (sampler :greedy t)
      (declare (ignore sampler))
      (princ (cl-llama-cpp:generate ctx "2 + 2 ="
                                    :max-tokens 16 :temp 0.0)))

    ;; Creative sampling
    (princ (cl-llama-cpp:generate ctx "Once upon a time"
                                  :max-tokens 128
                                  :temp 1.2
                                  :top-k 40
                                  :min-p 0.05))))
#+end_src

** Running Tests

Smoke tests run without a model:

#+begin_src bash
sbcl --eval '(ql:quickload "cl-llama-cpp/tests")' \
     --eval '(asdf:test-system "cl-llama-cpp/tests")' \
     --eval '(quit)'
#+end_src

Integration tests require a GGUF model:

#+begin_src bash
LLAMA_TEST_MODEL=/path/to/model.gguf \
LLAMA_TEST_EMBED_MODEL=/path/to/embedding-model.gguf \
sbcl --eval '(ql:quickload "cl-llama-cpp/tests")' \
     --eval '(asdf:test-system "cl-llama-cpp/tests")' \
     --eval '(quit)'
#+end_src
